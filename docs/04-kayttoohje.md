# 4. Käyttöohje

## Some-kanavan yhdistäminen

1. Avaa sovellus ja mene **Markkinointi**-osioon.
2. **Facebook:** klikkaa "Yhdistä Facebook" → Continue → valitse RNR Salonki -sivu.
3. **Instagram:** klikkaa "Yhdistä Instagram" → kirjaudu Instagram-tunnuksilla → Allow.
4. Yhdistetty kanava näkyy vihreänä **"Aktiivinen"**.

> Instagram toimii itsenäisesti — sitä ei tarvitse linkittää Facebookiin.

## Kanavan katkaiseminen

Yhdistetyn kanavan kohdalla on punainen **"Katkaise yhteys"** -nappi. Klikkaa sitä ja vahvista. Yhteys poistetaan ja voit yhdistää toisen tilin.

## Postauksen luominen

1. Klikkaa **"Uusi postaus"**.
2. Kirjoita teksti itse **tai** anna AI:n ehdottaa (kirjoita aihe → Sparkles-nappi).
3. Valitse kanava: Facebook, Instagram tai molemmat.
4. Tallenna luonnos. Luonnokset näkyvät **Postaukset**-välilehdellä.

## Viestit ja analytiikka

- **Viestit**-välilehti näyttää DM:t ja kommentit, ja merkitsee lukemattomat.
- **Analytiikka** näyttää tilastot kun kanavat ovat yhdistettyinä.

> Näiden täysi toiminnallisuus (oikea julkaisu, vastaaminen) rakennetaan seuraavaksi — katso [Seuraavat askeleet](./06-seuraavat-askeleet.md).
