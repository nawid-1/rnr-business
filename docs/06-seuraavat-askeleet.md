# 6. Seuraavat askeleet

## Markkinointi (jatko)

- [ ] Postausten **oikea julkaisu** Facebookiin ja Instagramiin (luonnos → julkaistu)
- [ ] Postausten **ajastus** (julkaise tiettynä aikana)
- [ ] **Viestien ja kommenttien haku** alustalle
- [ ] **Vastaaminen** DM:iin ja kommentteihin suoraan alustalta
- [ ] **Analytiikka** käyttöön (seuraajat, tykkäykset, kattavuus)

## Muut osiot

- [ ] **Asiakkaat (CRM):** asiakasrekisteri, varaukset, viestintä
- [ ] **Talous:** laskutus, kulut, raportit
- [ ] **AI Assistentti:** Claude kytketään hallitsemaan toimintoja (MCP)

## Myyntivaihe (SaaS)

- [ ] **Meta App Review** — jotta muutkin kuin testaajat voivat käyttää
- [ ] **Maksut (Stripe):** tilausmalli, super admin aina ilmainen
- [ ] **Käyttäjähallinta:** rekisteröityminen, kirjautuminen, roolit
- [ ] **Oma domain** (esim. rnrbusiness.com)
